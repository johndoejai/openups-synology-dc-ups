# Contributing

Thanks for considering a contribution. The goal is to keep this project lightweight and measurement-driven.

## How to contribute
- Use **GitHub Issues** for questions, bugs, and proposals.
- Submit **small PRs** (one change per PR).

## For tuning-related PRs
Include:
- Before/after KPIs:
  - top-ups/week (CHRG events)
  - peak battery voltage
  - time ≥ 13.8 V
  - PCB temp max
- Log snippet or weekly plot
- Exact OpenUPS settings used

## Style
- Keep docs concise and reproducible.
- Prefer copy/paste commands over long explanations.
